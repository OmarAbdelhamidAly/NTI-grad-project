# Chinook Database Test Suite

Use these 5 questions to validate the system's accuracy. They range from basic data counting to expert-level multi-join analysis.

---

### Question 1: Basic (Easy)
**Question**: "How many tracks are there in each genre? Sort from highest to lowest."
**Validated SQL**:
```sql
SELECT g.Name, COUNT(t.TrackId) as TrackCount 
FROM Track t 
JOIN Genre g ON t.GenreId = g.GenreId 
GROUP BY g.GenreId 
ORDER BY TrackCount DESC;
```

---

### Question 2: Intermediate (Normal)
**Question**: "List the top 5 customers who have spent the most money on purchases."
**Validated SQL**:
```sql
SELECT c.FirstName, c.LastName, SUM(i.Total) as TotalSpent 
FROM Customer c 
JOIN Invoice i ON c.CustomerId = i.CustomerId 
GROUP BY c.CustomerId 
ORDER BY TotalSpent DESC 
LIMIT 5;
```

---

### Question 3: Advanced (Medium)
**Question**: "Which artist has the most tracks in the 'Rock' genre?"
**Validated SQL**:
```sql
SELECT ar.Name, COUNT(t.TrackId) as RockTrackCount 
FROM Artist ar 
JOIN Album al ON ar.ArtistId = al.ArtistId 
JOIN Track t ON al.AlbumId = t.AlbumId 
JOIN Genre g ON t.GenreId = g.GenreId 
WHERE g.Name = 'Rock' 
GROUP BY ar.ArtistId 
ORDER BY RockTrackCount DESC 
LIMIT 1;
```

---

### Question 4: Complex (Hard)
**Question**: "Identify the employees who report to 'Andrew Adams' and calculate the total revenue generated by the customers they support."
**Validated SQL**:
```sql
SELECT e.FirstName, e.LastName, SUM(i.Total) as RevenueGenerated 
FROM Employee e 
JOIN Employee m ON e.ReportsTo = m.EmployeeId 
JOIN Customer c ON e.EmployeeId = c.SupportRepId 
JOIN Invoice i ON c.CustomerId = i.CustomerId 
WHERE m.FirstName = 'Andrew' AND m.LastName = 'Adams' 
GROUP BY e.EmployeeId 
ORDER BY RevenueGenerated DESC;
```

### Question 5: Expert (Very Hard)
**Question**: "Find the average track length (in milliseconds) for each album, but only for albums that have a total duration longer than 60 minutes."
**Validated SQL**:
```sql
SELECT a.Title, AVG(t.Milliseconds) as AvgTrackLength 
FROM Album a 
JOIN Track t ON a.AlbumId = t.AlbumId 
GROUP BY a.AlbumId 
HAVING SUM(t.Milliseconds) > 3600000 
ORDER BY AvgTrackLength DESC;
```
